```tsx
이 게시물은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.
```

## 링크 및 배너 유형

### 현재 사용 중인 방식

#### 1. Dynamic Banner (동적 배너) ✅ 구현 완료

- **Widget ID**: 938589 (Mid-Content), 938590 (End-Post), 938591 (Sidebar)
- **장점**:
  - 페이지 내용 자동 분석하여 관련 상품 추천
  - 별도 상품 선택 불필요
  - 반응형 디자인
- **단점**:
  - 외부 스크립트 로딩 (성능 영향 ~5-10점)
  - 카테고리 정확도는 직접 링크보다 낮음

**사용 위치**: 블로그 포스트 중간 및 끝 (`src/app/[locale]/posts/[slug]/page.tsx`)

### 추가 사용 가능한 방식

#### 2. 상품 링크 (Product Link)

- **장점**: 특정 상품 직접 추천 시 높은 전환율
- **단점**: 상품 재고 소진 시 링크 무효화
- **권장 사용처**: 제품 리뷰 포스트

#### 3. 간편 링크 (Simple Link)

- **장점**: 쿠팡 메인/카테고리 페이지 연결, 유지보수 불필요
- **단점**: 전환율 낮음
- **권장 사용처**: 범용 추천 링크

#### 4. 검색 위젯 (Search Widget)

- **장점**: 사용자가 직접 검색 가능, 높은 전환율
- **단점**: 페이지 레이아웃 영향
- **권장 사용처**: 쇼핑몰형 사이트

#### 5. Partners API

- **장점**: 프로그래매틱 상품 검색 및 추천 가능
- **단점**: 개발 복잡도 높음
- **권장 사용처**: 대규모 자동화된 콘텐츠 생성

**향후 검토**: API 활용하여 AI 리포트 카테고리별 맞춤 상품 추천

## 금지된 활동

### ⛔️ 절대 금지 (계정 정지 사유)

#### 1. 지식재산권 침해
- ❌ 쿠팡 로고/상표 무단 사용
- ❌ "쿠팡과 제휴", "공식 파트너십" 등 오해 유발 표현
- ✅ **허용**: "쿠팡 파트너스 활동" (사실 진술)

#### 2. 자동실행 스크립트
```javascript
// ❌ 금지된 코드 예시
window.onload = function() {
  window.location.href = "https://coupa.ng/xxxxx"; // 자동 리다이렉트
}

setTimeout(() => {
  document.getElementById('coupang-link').click(); // 자동 클릭
}, 3000);
```

#### 3. 플로팅 배너
- ❌ `position: fixed` + 스크롤 따라다니는 배너
- ❌ 팝업 형태로 배너 강제 노출
- ✅ **허용**: 정적 배너 (본문 내 삽입)

#### 4. 타이머/긴급 문구
- ❌ "오늘만 50% 할인!"
- ❌ "남은 시간: 03:45:12"
- ✅ **허용**: 사실 기반 정보 ("현재 가격: ₩29,900")

#### 5. 성인/불법/허위 콘텐츠
- ❌ 성인용품, 도박, 불법 의약품
- ❌ 허위/과장 리뷰
- ✅ **허용**: 정직한 리뷰 및 사용 후기

### 현재 구현 검증

✅ **통과**: 위 금지 사항 모두 준수 중

## 마케팅 전략

### 공식 권장사항 (쿠팡 파트너스 가이드 pp.72-77)

#### 1. 카테고리 선택 (높은 수익률 카테고리)

| 카테고리 | 평균 수수료율 | 권장 이유 |
|----------|--------------|----------|
| **전자기기** | 1.0~3.0% | 단가 높음, 구매 전환율 높음 |
| **생활용품** | 3.0~5.0% | 반복 구매 많음 |
| **패션의류** | 5.0~7.0% | 시즌성 강함, 높은 수수료 |
| **식품** | 3.0~4.0% | 구매 빈도 높음 |

**colemearchy-blog 적합 카테고리**:
- ✅ 전자기기 (키보드, 마우스, 모니터)
- ✅ 도서 (개발서적, AI 관련 도서)
- ✅ 사무용품 (노트북 거치대, 의자)

#### 2. 키워드 전략

**특정 키워드 (Specific)** vs **범용 키워드 (Broad)**

| 키워드 유형 | 예시 | 전환율 | 경쟁도 |
|------------|------|--------|--------|
| **특정** | "맥북 프로 M3 16인치 리뷰" | ⭐⭐⭐⭐⭐ | 낮음 |
| **범용** | "노트북 추천" | ⭐⭐ | 높음 |

**권장 접근**:
1. 롱테일 키워드 공략: "AI 개발용 키보드 추천"
2. 구체적 문제 해결: "맥북 외장 모니터 연결 방법"
3. 비교 콘텐츠: "LG vs 삼성 모니터 비교"

#### 3. 조회수 최적화

**제목 작성 원칙**:
- ✅ 구체적 숫자 포함: "개발자가 추천하는 키보드 Top 5"
- ✅ 문제 해결 지향: "맥북 발열 줄이는 방법 3가지"
- ❌ 클릭베이트: "이거 모르면 후회합니다" (신뢰도 하락)

**썸네일**:
- 고해상도 이미지 사용
- 텍스트 오버레이 최소화
- 제품 실사용 사진 선호

#### 4. 전환율 최적화

**CTA (Call-to-Action) 배치**:
- ✅ 본문 중간: 정보 제공 후 자연스럽게 배너 삽입
- ✅ 포스트 끝: 독자가 다음 행동을 고민할 때 노출
- ❌ 본문 시작 직후: 정보 제공 전 광고는 이탈률 증가

**현재 구현**:
```
📍 Mid-Content: 본문 50% 지점
📍 End-Post: 포스트 끝
```

**콘텐츠 품질**:
- ✅ 실사용 후기 (신뢰도 ↑)
- ✅ 장단점 균형 있게 서술
- ❌ 무조건 긍정적 리뷰 (의심 ↑)

#### 5. 꾸준한 콘텐츠 발행

**권장 주기**:
- 최소: 주 1회
- 권장: 주 2~3회
- 이상적: 매일

**colemearchy-blog 현재 상태**:
- ✅ YouTube 자동 동기화: 매일 새 영상 → 블로그 포스트 자동 생성
- ✅ AI 리포트 자동 생성: 매일 3개 포스트
- 🎯 목표: 일 5개 이상 콘텐츠 발행
